# PhotoOrganaize
はじめに。

写真を整理するためのスクリプト置き場です。
本リポジトリのスクリプトを使用した場合に発生した問題について、Commeの実は一切の責任を負わないものとします。
自分の環境で動けばOKで作っていますのでコードの美しさはおろか、例外系処理は一切考慮していません。
使用前のバックアップは必ず取得してください。

コピペはご自由にどうぞ。使用・コピペ報告などは特に不要です。

-----------------------------
【リポジトリないのスクリプト。単体で動きます】
・fileFuriwakeLastModDate.vbs
ファイルの最終更新日をもとに振り分けします。

・fileFuriwakeExif.vbs
Exifデータをもとに振り分けします。実行速度が少し遅めです。
